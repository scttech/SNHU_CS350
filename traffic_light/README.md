# Traffic Light Project

<img src="./images/common/traffic_light.png" alt="Breakout board placement" width="100" />

This project is a simple implementation of a traffic light system using Python.

It is a simple project based off the curriculum from SNHU's CS-350 course.  This is a great project
for continuing your learning of Python and the Raspberry Pi.

* [Module 1](./module1/README.md) of the project focuses on setting up the software and getting the green LED to turn on.
* [Module 2](./module2/README.md) of the project goes through initial coding the traffic light state machine for the green 
* [Module 3](./module3/README.md) of the project focuses on `pytest` so that we can develop an understanding of testing 
and of state machines
