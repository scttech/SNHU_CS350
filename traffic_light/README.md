# Traffic Light Project

<img src="./images/common/traffic_light.png" alt="Traffic Light" width="100" />

This project is a simple implementation of a traffic light system using Python.

It is a simple project based on the curriculum from SNHU's CS-350 course.  This is a great project
for continuing your learning of Python and the Raspberry Pi.

* [Module 1](./module1/README.md) focuses on setting up the software and getting the green LED to turn on.
* [Module 2](./module2/README.md) goes through initial coding the traffic light state machine for the green 
* [Module 3](./module3/README.md) focuses on `pytest` so that we can develop an understanding of testing 
and of state machines
* [Module 4](./module4/README.md) goes through setting up the yellow LED and testing it
* [Module 5](./module5/README.md) goes through setting up the red LED
* [Module 6](./module6/README.md) expands on the state machine setup in Module 2
* [Module 7](./module7/README.md) creating state diagrams
* [Module 7](./module7/README.md) introduces Docker
* [Module 8](./module8/README.md) setting up the MQTT server in Docker
